Toast App Configuration Guide

This document explains how Toast App stores your settings and what each configuration option means. While most users won't need to edit the configuration file directly, understanding these options can help you get the most out of Toast App.

Configuration Overview

Toast App stores your settings in a JSON file. The file contains all your preferences, button configurations, and appearance settings. While Toast App's Settings window provides an easy way to change most options, some advanced users might want to edit the configuration file directly.

The configuration file includes:

  • Your global hotkey (the keyboard shortcut that opens Toast App)
  • Your pages and buttons
  • Visual appearance settings (theme, position, size)
  • Advanced behavior settings (auto-launch, hiding behavior)
  • Subscription information

Configuration File Location

The configuration file is stored in a different location depending on your operating system:

macOS

~/Library/Application Support/toast-app/config.json

Windows

%APPDATA%\toast-app\config.json

Linux

~/.config/toast-app/config.json

The file is automatically created when you first launch Toast App. If the file is accidentally deleted or corrupted, Toast App will create a new file with default settings.

Global Settings

These settings affect the overall behavior of Toast App.

globalHotkey

String

The keyboard shortcut that triggers Toast App from anywhere on your computer.

Default Value:

"Alt+Space"

Examples of valid hotkeys:

  • "Alt+Space" - Press Alt and Space together
  • "Ctrl+Shift+T" - Press Ctrl, Shift, and T together
  • "CommandOrControl+P" - Press Command (Mac) or Control (Windows) and P together

Note: If your chosen hotkey conflicts with another application, you'll need to choose a different one.

firstLaunchCompleted

Boolean

Indicates whether you've completed the initial setup process. This is set automatically and you generally don't need to change it.

Default Value:

false

Pages and Buttons

Toast App organizes buttons into pages, allowing you to group related actions together.

Pages

Each page has the following properties:

name

String

The name of the page, shown in the tab at the top of the Toast popup.

Example: "Applications", "Development", "Web Links"

shortcut

String

The keyboard shortcut to switch to this page (typically a number from 1-9).

Example: "1", "2", "3"

buttons

Array

A list of buttons that appear on this page.

Buttons

Each button has the following properties:

name

String

The name of the button, shown on the button in the Toast popup.

Example: "Terminal", "Chrome", "Slack"

shortcut

String

The keyboard shortcut to trigger this button (typically a single letter or symbol).

Example: "T", "C", "S"

icon

String

An emoji or icon to display on the button.

Example: "‚å®Ô∏è", "üåê", "üìÅ"

action

String

The type of action this button performs. Must be one of: "exec", "open", "shortcut", "script", or "chain".

Each button also includes additional properties depending on its action type. See Button Actions for details.

Button Actions

Toast App supports several types of actions for buttons. Each action type requires different additional properties.

Execute Command (exec)

Runs a shell command on your computer.

command

String

The shell command to execute.

Example: "open -a Terminal" (macOS) or "start cmd" (Windows)

workingDir

String

The directory in which to run the command (optional).

Example: "~/Documents"

runInTerminal

Boolean

Whether to run the command in a visible terminal window (optional).

Default Value:

false

Open URL/File (open)

Opens a URL in your default browser or a file/folder with its associated application.

url

String

The URL to open in a browser.

Example: "https://www.google.com"

path

String

The file or folder path to open.

Example: "~/Documents" or "/path/to/document.pdf"

application

String

The specific application to use for opening (optional).

Example: "Firefox" or "Microsoft Word"

Keyboard Shortcut (shortcut)

Simulates pressing a keyboard shortcut.

keyShortcut

String

The keyboard shortcut to simulate.

Example: "Ctrl+C", "Alt+Tab", "Cmd+Space"

Custom Script (script)

Executes a custom script in various languages.

script

String

The script content to execute.

scriptType

String

The language of the script. Must be one of: "javascript", "applescript" (macOS only), "powershell" (Windows only), or "bash" (macOS/Linux only).

Action Chain (chain)

Executes multiple actions in sequence.

actions

Array

A list of actions to execute in sequence.

stopOnError

Boolean

Whether to stop executing actions if one fails.

Default Value:

true

Appearance Settings

These settings control how Toast App looks and behaves visually.

theme

String

The visual theme for Toast App.

Default Value:

"system"

Options:

  • "light" - Always use light theme
  • "dark" - Always use dark theme
  • "system" - Follow your system's light/dark mode setting

position

String

Where the Toast popup appears on your screen.

Default Value:

"center"

Options:

  • "center" - Center of the screen
  • "top" - Top center of the screen
  • "bottom" - Bottom center of the screen
  • "cursor" - Near your mouse cursor

size

String

The size of the Toast popup window.

Default Value:

"medium"

Options:

  • "small" - 350x400 pixels
  • "medium" - 400x500 pixels
  • "large" - 500x600 pixels

opacity

Number

The transparency of the Toast popup (0.1 to 1.0, where 1.0 is fully opaque).

Default Value:

0.95

buttonLayout

String

How buttons are arranged in the Toast popup.

Default Value:

"grid"

Options:

  • "grid" - Buttons arranged in a grid (default)
  • "list" - Buttons arranged in a vertical list

Advanced Settings

These settings control Toast App's behavior.

launchAtLogin

Boolean

Whether Toast App starts automatically when you log in to your computer.

Default Value:

false

hideAfterAction

Boolean

Whether the Toast popup closes automatically after you execute an action.

Default Value:

true

hideOnBlur

Boolean

Whether the Toast popup closes when it loses focus (e.g., when you click elsewhere).

Default Value:

true

hideOnEscape

Boolean

Whether the Toast popup closes when you press the Escape key.

Default Value:

true

showInTaskbar

Boolean

Whether the Toast App icon appears in the taskbar/dock when the popup is open.

Default Value:

false

Subscription Settings

These settings relate to your Toast App subscription status.

isSubscribed

Boolean

Whether you have an active subscription.

Default Value:

false

subscribedUntil

String

The date until which your subscription is valid (ISO format).

Default Value:

""

pageGroups

Number

The number of page groups you can create based on your subscription level.

Default Value:

1

Page limit by subscription:

  • Free users: 1 page
  • Authenticated users: 3 pages
  • Subscribers: 9 pages

Backup and Restore

You can back up and restore your Toast App configuration to transfer settings between computers or recover from issues.

Exporting Configuration

To export your configuration:

  1. Open the Settings window in Toast App
  2. Go to the Advanced tab
  3. Click "Export Configuration"
  4. Choose where to save the configuration file

Importing Configuration

To import a configuration:

  1. Open the Settings window in Toast App
  2. Go to the Advanced tab
  3. Click "Import Configuration"
  4. Select the configuration file to import
  5. Choose whether to replace or merge with your existing configuration

Automatic Backups

Toast App automatically creates a backup of your configuration before making changes. The backup is stored in the same location as your configuration file with a .backup extension.
